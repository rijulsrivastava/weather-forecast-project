<h1>Weather Forecast Project</h1>
<p>In this project HTML, TailwindCSS is used make static layout of weather forecasting webpage  Javascript is used to make webpage dynamic and responsive.
    It uses OpenWeatherMap API for forcasting and it offers various features such as location based forcasting, 5-day forecast, current weather forecast.
</p>

<h2> Objectives of this project are:</h2>
<ol>
    <li>To develop Weather forcasting webpage</li>
    <li>Fetching and integrating OpenWeatherMap API</li>
    <li>To have gps based forecasting</li>
    <li>To have forecasting by city name</li>
    <li>To have 5-day forecast</li>
    <li>To have a functional Celsius to Fahrenheit toggle button</li>
    <li>To have custom allert function for etreme temperature</li>
    <li>To store recently searched city in browser's local storage</li>
    <li>To handle errors by validating user input</li>
</ol>
